# InPost Pay for Magneto 2 Commerce
Module modifies InPost_InPostPay module to support Magento 2 Commerce.

### Compatibility
- Magento Commerce 2.4.6
- Not tested in other versions

### Installation
To enable this extension run following commands:
- `bin/magento module:enable InPost_InPostPayCommerce`
- `bin/magento setup:upgrade`
